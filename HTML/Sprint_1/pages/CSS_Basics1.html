LET'S MAKE OUR WEBSITE COLORFUL

In this blog post you will learn how to make your website colorful. First you need to create in VS Code CSS file. Now when you open it, we can add some backround colors: 

           
               body {
                   background-color: whitesmoke;
               }

If you want to change fonts, you can add like this:

            
                body or p (here you write which tag's font you want to change) {
                    font-family: century gothic
                }
